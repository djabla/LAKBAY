-------------------------
LAKBAY 2D ADVENTURE GAME
-------------------------

FOR INSTALLATION:
   1. Unzip the demo version zip file
   2. Open the folder and find LAKBAY Adventure Game.exe to run the game
   3. To close the game, click the exit on the top right of the screen

*******************************************************

GAME CONTROLS:
   Walk Left - D
   Walk Right - A
   Jump - W
   Shoot - G
   Pause Game - P
